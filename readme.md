# Frontend Templates

A collaborative repository of reusable frontend templates built by our team.

## Tech Stack

- React (Vite)
- HTML, CSS, JavaScript
- Tailwind / CSS Modules

## Folder Structure

- react/ → React-based templates
- html-css/ → Static templates
- shared-components/ → Reusable UI components

## How to Use

1. Clone repo
2. Go to desired template folder
3. Install dependencies (if any)
4. Run project
